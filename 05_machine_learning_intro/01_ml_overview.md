# 🤖 Machine Learning Overview

## 📌 What is Machine Learning?

Machine Learning = Teaching computers to learn patterns from data.

---

## 🧠 Types of ML

### 1. Supervised Learning
- Data has labels
- Example:
  - Predict house price
  - Classify spam email

---

### 2. Unsupervised Learning
- No labels
- Example:
  - Customer segmentation

---

## 🔁 ML Workflow

1. Collect Data
2. Clean Data
3. Train Model
4. Evaluate Model
5. Make Predictions

---

## 🎯 Key Idea

> Model learns a function:

y = f(X)

---

## 🔥 Examples

| Problem | Model |
|--------|------|
| Price Prediction | Linear Regression |
| Spam Detection | Logistic Regression |
