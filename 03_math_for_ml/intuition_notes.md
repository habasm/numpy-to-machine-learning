# 🧠 Math Intuition for Machine Learning

---

## 📌 Vectors

- A vector = list of features
- Example:
  - height, weight → [170, 65]

👉 In ML:
Vectors represent data points

---

## 📌 Matrices

- Matrix = dataset
- Rows = samples
- Columns = features

👉 In ML:
X = dataset

---

## 📌 Dot Product

- Measures similarity

👉 In ML:
Used in predictions:
y = Xw

---

## 📌 Mean

- Average value

👉 In ML:
Used in loss functions

---

## 📌 Variance

- Spread of data

👉 High variance = unstable model
